# upgraded-funicular

## Descriptive Variable and Function Naming Examples

This repository demonstrates best practices for naming variables, functions, and classes in code. It provides side-by-side comparisons of poorly named code versus well-named code to illustrate the importance of descriptive naming.

## Contents

- **example_poor_naming.py** - Examples of code with unclear, abbreviated, and single-letter names
- **example_good_naming.py** - The same functionality refactored with clear, descriptive names
- **NAMING_GUIDE.md** - Comprehensive guide to variable and function naming best practices

## Quick Start

View the examples:

```bash
# See the difference in action
python example_poor_naming.py
python example_good_naming.py
```

Both scripts produce identical output, but the code quality and readability differ significantly.

## Why This Matters

Good naming practices:
- Make code self-documenting
- Reduce the need for comments
- Help prevent bugs
- Speed up code reviews
- Ease onboarding of new team members
